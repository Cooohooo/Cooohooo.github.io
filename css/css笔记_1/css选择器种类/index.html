          # CSS选择器种类

          # CSS 基本选择器

          • * 选择所有元素
          • #p 选择id为p的元素
          • .p 选择所有class包含p的元素

          # CSS 组合选择器：

          • div, p 选择所有div元素和p元素
          • div p 选择div元素下的所有p元素
          • div > p 选择所有父级是div元素的p元素
          • div + p 选择紧接着div元素之后的相邻的同级p元素
          代码实现：
<!DOCTYPE html>
<html>
    <head>
    <style>
        div+p
        {
            background-color: red;
        }
    </style>
    </head>
    <body>
        <h1>欢迎来到我到的主页</h1>
        <div>
            <p>我是唐老鸭。</p>
            <p>我住在 Duckburg。</p>
        </div>
        <p>我最好的朋友是米老鼠。</p> (这个位置的p元素会变成red，p是div之后的同级的元素)
        <p>我的样式不会改变。</p>
    </body>
</html>
          • p ~ ul 选择p元素之后的同级ul元素
          代码实现：
<!DOCTYPE html>
<html>
    <head>
    <style> 
        p~ul
        {
            background:#ff0000;
        }
    </style>
    </head>
    <body>
        <div>一个 div 元素。</div>
        <ul>
            <li>咖啡</li>
            <li>牛奶</li>
            <li>茶</li>
        </ul>
        <p>第一段。</p>
        <ul>
            <li>咖啡</li> (这三个li会变成红色)
            <li>牛奶</li>
            <li>茶</li>
        </ul>
        <h2>另一个列表</h2>
        <ul>    
            <li>咖啡</li> (这三个li也会变成红色)
            <li>牛奶</li>
            <li>茶</li>
        </ul>
    </body>
</html>

          # CSS 伪选择器：

          • a: hover 选择鼠标指针浮动在其上的的元素
          • input: focus 选择获得焦点的input元素
          • input: checked 选择器匹配每个选中的输入元素（仅适用于单选按钮或复选框）

          • p: before 选择器在伪元素的真正元素内部之前添加新内容
          • p: after 选择器在伪元素的真正元素内部之后添加新内容

          # 伪元素:

          before和:after添加的内容默认是inline元素；这个两个伪元素的content属性，表示伪元素的内容,设置:before和:after时必须设置其content属性，否则伪元素就不起作用.

          • 字符串：字符串作为伪元素的内容添加到主元素中

          注意：字符串中若有html字符串，添加到主元素后不会进行html转义，也不会转化为真正的html内容显示，而是会原样输出

          • attr (attr_name)：伪元素的内容跟主元素的某个属性值进行关联，及其内容为主元素的某指定属性的值
          • url()/uri(): 引用外部资源，例如图片；


          • : not(p) 设置非<p>元素的所有其他元素的属性
          :not(p)
{ 
    background-color: #ff0000; （设置除了<p>元素之外的其他的背景颜色）
}
