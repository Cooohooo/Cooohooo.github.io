            # Margin 用法：

            # 具体问题

            利用 margin 外部尺寸实现等高布局，此布局多出现在分栏有背景色或者中间有分割线的布局中，由于height = 100%需要在父级设定高度时才有效，因此可以使用margin负值实现

            HTML:

            <div id="colLeft" class="column-left">
    <h4>正方观点</h4>
    <p>观点1</p>
</div>
<div id="colRight" class="column-right">
    <h4>反方观点</h4>
    <p>观点1</p>
</div>

<input type="button" id="leftMore" value="更多正方观点">
<input type="button" id="rightMore" value="更多反方观点">

            CSS:

            .column-left,
.column-right {
    margin-bottom: -9999px;
    padding-bottom: 9999px;
}
.column-left {
    background-color: #34538b;
}
.column-right {
    background-color: #cd0000;
}

            # margin合并：

            • 相邻兄弟元素margin合并
            p{ margin: 1em 0;}
<p> first line </p>
<p> second line </p>
            • 父级的第一个/最后一个子元素
            <div class = 'father'>
    <div class = 'son' style='margin-top:80px;'></div>
</div>

<div class = 'father' style='margin-top:80px;'>
    <div class = 'son'></div>
</div>

<div class = 'father' style='margin-top:80px;'>
    <div class = 'son' style='margin-top:80px;'></div>
</div>
            • 空块级元素的margin合并
            .father{ overflow: hidden}
.son{ margin: 1em 0;}
<div class=“father”>
    <div class=“son”></div>
</div>

            # margin合并计算规则：

            • 正正取最大值
            • 正负值相加
            • 负负最负值

            # margin: auto 用法：

            • 如果一侧定值，一侧auto，则auto为剩余空间大小
            • 如果两侧都是auto，则平分剩余空间

            所以块状元素右对齐的方法不止 float: right，很多时候，margin-right: auto也是很好的方法，margin的auto属性计算就是为块级元素左中右对齐而设计的，和内联元素(inline-element)的text-align控制左中右正好呼应

            举个栗子 CSS如下：

            .son{
    width: 200px;
    margin-right: auto;
    margin-left: auto;
}

            垂直方向的margin居中其实也是有方法的

            • 使用writing-mode改变文档流方向
            .father{
    height: 200px;
    writing-mode: vertical-lr;
}
.son{
    height: 100px:
    margin: auto;
}

            这样就可以垂直居中对齐，但是这样就不能水平居中了

            • 使用绝对定位元素的margin: auto居中
            .father{
    width: 300px; height: 150px;
    position: relative;
}
.son{
    position: absolute;
    top: 0; right: 0; left: 0; bottom: 0;
    width: 200px; height: 100px;
    margin: auto;
}

            # 其他

            CSS incline-block 元素 特性展示

            每一个元素都有两个盒子，inline-block 指的是外在的盒子是 “内联盒子(用于和文字在一行中显示)”，内在的盒子是“块级盒子(用于显示块级元素，e.g table list)”

            HTML：                            
和文字平起平坐的表格：
<div class="inline-table">
    <p>第1列</p>
    <p>第2列</p>
</div>

            CSS：
.inline-table {
    display: inline-table;
    width: 128px;
    margin-left: 10px;
    border: 1px solid #cad5eb;
}
.inline-table > p {
    display: table-cell;
}

            效果展示：（一个table 原本是block元素，但是通过inline-table，把他变成了可以和文字在同一排展示的table）
