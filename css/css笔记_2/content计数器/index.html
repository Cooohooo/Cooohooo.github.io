        # Content 计数器：

        # 定义

        Css 计数器跟军训报数一样，其中有这么几个关键点：

        1. 班级命名：有个称呼，如机械4班
        2. 报数规则：1 2 3 4递增报数，还是1 2 1 2 循环报数
        3. 开始报数：何时开始报数的行为

        以上三个点对应了CSS计数器的两个属性（counter-reset 和 counter-increment）和一个方法（counter() / counters()）

        • counter-reset：“计数器-重置”的意思，就是上面的班级命名，主要作用给计数器命名，并通知计数器从哪个数字开始计时，默认是 0；counter-reset 的值可以是负数，比如 -2，counter-reset 还可以同是给多个计数器命名，e.g：直接空格分开，不需要使用逗号分隔
        .xxx{counter-reset: wangxiaoer 2 wangxiaosan 3;}
        • counter-increment：“计数器递增”的意思，值为counter-reset的一个或多个关键字，后面可跟随数字，表示每次计数的变化，如果省略，则使用默认值 1
        .counter{
    counter-reset: wangxiaoer 2;
    counter-increment: wangxiaoer
}
.counter:before{
    content: counter(wangxiaoer);
}
<p class=“counter”></p>

        counter-increment 选中的是 <p>元素，counter-reset值增加，默认递增1，于是从2 --> 3，wangxiaoer就是这里的计数器

        # counter-increment也可以设置在伪元素上面：

        .counter{
    counter-reset: wangxiaoer 2;
}
.counter:before{
    content: counter(wangxiaoer);
    counter-increment: wangxiaoer
}
<p class=“counter”></p>

        全局的计数器加1，所显示的数字依然是 3

        # counter-increment作用在父元素和子元素上面：

        .counter{
    counter-reset: wangxiaoer 2;
    counter-increment: wangxiaoer
}
.counter:before{
    content: counter(wangxiaoer);
    counter-increment: wangxiaoer
}
<p class=“counter”></p>

        counter-increment 选中的是

        元素，父元素一次，子元素一次，于是从2 --> 4，wangxiaoer就是这里的计数器，无论在何处，只要有counter-increment，对应的计数器的值就会变化，counter()只是输出

        • 方法 counter/counter() 这是方法，不是属性，这里的作用很简单，即显示计数，常用的名称和用法如下：

        counter(name, style) ————counter方法的语法

        style支持的关键字是list-style-type支持的那些值，他的作用是：我们递增递减的不一定是数字还可以是英文字母或罗马文等

        举个栗子：

        核心CSS代码如下：

        content: counter(wangxiaoer, lower-roman);

        counter还支持级联，也就是说一个content属性值可以有多个counter()方法

        举个栗子：

        核心CSS代码如下：

        .counter{    counter-reset: wangxiaoer 2 wangxiaosan 3; }
.counter:before{
    content: counter(wangxiaoer);
    counter-increment: wangxiaoer
